Internet Component Suite (ICS) (ID #17323)


OverByte

Component collection for Internet applications. Support most TCP/IP protocols such as TCP client, TCP server, UDP client, UDP server, TELNET client, TELNET server, FTP client, FTP server, NNTP client, HTTP client, HTTP server (WEB server), SMTP client, POP3 client, PING, ICMP, FINGER, VT100 terminal emulation, MIME decoding, mail file attach, client/server and more. Full source code with numerous sample programs. Free support using a mailing list. Works with all Delphi and BCB versions.
